ETO Newsletter Sender 사용설명서

월간 한국어 뉴스레터 발송 프로그램 · 2026년 7월 기준

1. 이 프로그램이 하는 일

구독자 명단(CSV 파일)을 불러와서 월간 뉴스레터 이메일을 보내는 프로그램입니다. 실수로 잘못 발송되는 일을 막기 위해, 발송 전에 반드시 ① 명단 대조 ② 테스트 메일 확인 ③ 확인 체크를 거치도록 되어 있고, 마지막 순간에도 5초 동안 취소할 수 있습니다.

프로그램 설치와 초기 설정은 담당자가 해 드립니다. 이 설명서는 설치가 끝난 뒤 실제로 사용하는 방법만 다룹니다.

2. 프로그램 시작하기

바탕화면(또는 프로그램 폴더)의 Update ETO Newsletter 아이콘을 더블클릭하세요. 최신 버전을 자동으로 내려받은 뒤 프로그램이 열립니다. 인터넷이 안 될 때는 Run ETO Newsletter를 대신 실행하면 업데이트 없이 바로 열립니다.

3. 매달 발송하는 순서

  1. ① 명단 열기 — 왼쪽 위 Upload list… 버튼을 눌러 이번 달 구독자 CSV 파일을 선택합니다. 표에 명단이 나타나고 위쪽에 총 인원이 표시됩니다.
  2. ② 명단 대조 — Compare with MailerLite 버튼을 누릅니다. 지난달 발송 명단과 비교해서 새로 들어온 사람 / 빠진 사람 / 수신거부한 사람을 보여줍니다. 내용을 확인하고 닫으면 체크리스트에 ✓가 켜집니다. (자세한 활용은 4장)
  3. ③ 뉴스레터 파일 선택 — 오른쪽 Browse… 버튼으로 이번 호 Word 파일(.docx)을 선택하고, 홈페이지에 올려 둔 PDF 주소를 PDF link 칸에 붙여넣습니다.
  4. ④ 미리보기 — Preview 버튼을 누르면 실제 발송될 이메일 모습이 인터넷 브라우저로 열립니다. 제목 띠, 본문, PDF 버튼, 맨 아래 배너까지 눈으로 확인하세요.
  5. ⑤ 제목 확인 — Subject 칸의 제목을 이번 달에 맞게 고칩니다. (지난번 제목이 남아 있으니 월만 바꾸면 됩니다.)
  6. ⑥ 테스트 메일 — Test email(s) 칸에 본인 이메일 주소를 넣고 Send test를 누릅니다. 여러 명이 확인하려면 주소를 쉼표(,)로 이어 적으세요. 받은 메일을 컴퓨터와 휴대폰에서 열어 이상이 없는지 확인합니다.
  7. ⑦ 확인 체크 — 대조(②)와 테스트(⑥)가 모두 끝나면 "I reviewed the comparison and the test email" 체크박스가 켜집니다. 내용에 문제가 없으면 체크하세요. 그래야 발송 버튼이 눌리게 됩니다.
  8. ⑧ 발송 —
    • Send to all subscribers를 누르면 표의 전원이 선택됩니다. 이번에 빼고 싶은 사람이 있으면 체크를 해제하세요.
    • 같은 버튼(Confirm — send now)을 한 번 더 누르고 확인 창에서 Yes를 누릅니다.
    • 버튼이 빨갛게 바뀌며 5초 카운트다운이 시작됩니다. 마음이 바뀌면 그 빨간 버튼을 누르면 취소됩니다. 5초가 지나면 발송됩니다.
발송이 끝나면 "The newsletter has been sent." 안내가 뜨고, MailerLite의 구독자 그룹이 이번 발송 명단으로 자동 정리됩니다. 이것으로 이번 달 작업은 끝입니다.

4. 명단 대조 창 활용하기

항목뜻할 수 있는 일
New sign-ups
(신규)		이번 CSV에 새로 들어온 사람 체크하고 Add checked to “ETO Korea Newsletter Subscribers”를 누르면 MailerLite 그룹에 미리 등록됩니다.
Departed
(빠짐)		지난달에는 있었는데 이번 CSV에서 빠진 사람 실수로 빠진 거라면: 체크 후 Add checked to the current list — 명단에 다시 들어오고, 수정된 CSV를 저장하라는 창이 뜹니다.
정말 정리할 거라면: 체크 후 Delete checked from MailerLite…
Unsubscribed
(수신거부)		이메일의 수신거부 링크를 직접 누른 사람 확인만 하면 됩니다. CSV에 남아 있어도 이 사람들에게는 자동으로 발송되지 않습니다. 다시 받게 하려고 임의로 조작하지 마세요 (본인 동의가 필요합니다).

한 번에 한 가지 작업을 하고 창이 닫힙니다. 다른 작업도 하려면 Compare with MailerLite를 다시 누르면 됩니다.

5. 뉴스레터 Word 파일 만들 때 요령

  • 제목은 글자를 크게 키우지 말고 Word 위쪽의 스타일 갤러리에서 "제목 1", "제목 2"를 골라 지정하세요. 그래야 이메일에서도 제목으로 예쁘게 표시됩니다.
  • 굵게, 기울임, 글머리 기호(•), 표는 이메일에 그대로 반영됩니다.
  • 문서에 붙여넣은 사진·그림은 받는 사람에게 안 보이는 경우가 많습니다. (Gmail, Outlook이 차단) 그림이 있으면 프로그램이 경고를 띄우니, 그때는 담당자와 상의하세요.
  • 상단 남색 제목 띠, PDF 다운로드 버튼, 맨 아래 SNS·수신거부 배너는 프로그램이 자동으로 붙입니다. 본문 내용만 쓰시면 됩니다.

6. 이런 메시지가 나오면

상황대처
발송 버튼이 회색이고 안 눌림고장이 아닙니다. 명단 대조(②)와 테스트 메일(⑥)을 마치고 확인 체크박스(⑦)를 켜면 눌리게 됩니다.
테스트 ✓가 갑자기 꺼짐테스트 후에 Word 파일이나 제목을 고쳤기 때문입니다. 바뀐 내용으로 테스트 메일을 다시 보내면 됩니다.
"The PDF link does not work"PDF 주소가 틀렸거나 아직 홈페이지에 올라가지 않은 것입니다. 주소를 확인해서 고치세요. 아무것도 발송되지 않았으니 안심하세요.
"Possible duplicate send" (노란 경고)같은 제목으로 최근에 이미 발송한 기록이 있다는 뜻입니다. 이번 달 것을 벌써 보냈는지 확인하고, 두 번 보내는 게 아니라고 확실할 때만 Yes를 누르세요.
"SAFETY STOP" (빨간 에러)받는 사람 설정이 확인되지 않아 프로그램이 발송을 스스로 중단한 것입니다(오발송 방지 장치). 발송되지 않았습니다. 담당자에게 알려주세요.
업데이트가 실패함인터넷 연결을 확인하고 Update ETO Newsletter를 다시 실행하세요. 그래도 안 되면 담당자에게 연락하세요.
"Authentication failed" / 연결 오류담당자에게 연락하세요. (설정 문제로, 매니저님이 고칠 일이 아닙니다.)

7. 주의사항

  • 프로그램 폴더 안의 파일을 옮기거나 지우지 마세요. (구독자 CSV 파일을 넣고 빼는 것은 괜찮습니다.)
  • 구독자 CSV와 config.json(설정 파일)은 개인정보·열쇠에 해당합니다. 이메일이나 메신저로 보내지 마세요.
  • MailerLite 홈페이지에 직접 들어가서 그룹을 지우지 마세요 — 다음 달 명단 대조가 안 됩니다.
  • 궁금하거나 이상하면 발송 전에 담당자에게 먼저 물어보세요. 발송은 되돌릴 수 없습니다.
