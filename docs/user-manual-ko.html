ETO Newsletter Sender 사용설명서

월간 한국어 뉴스레터 발송 프로그램 · 2026년 7월 기준

1. 개요

구독자 명단(CSV 파일)을 불러와 MailerLite를 통해 월간 뉴스레터 이메일을 발송하는 Windows 프로그램입니다. 오발송을 막기 위해 발송 전에 반드시 ① 명단 대조 ② 테스트 메일 ③ 확인 체크를 거쳐야 하며, 최종 발송 직전에도 5초 동안 취소할 수 있습니다.

화면 구성

영역내용
왼쪽 표업로드한 구독자 명단. 검색창으로 이름·회사·이메일을 찾을 수 있고, 열 제목을 누르면 정렬됩니다. 의심 주소(봇 가입 등)는 주황색으로 표시됩니다.
오른쪽 Campaign 패널뉴스레터 파일·PDF 링크·제목·테스트 메일· 발송 전 체크리스트·발송 버튼.
아래 상태 표시줄진행 상황과 안내 메시지가 표시됩니다.

2. 프로그램 시작

  • Update ETO Newsletter.bat 더블클릭 — 최신 버전을 자동으로 받은 뒤 실행됩니다. 평소에는 이것을 사용하세요.
  • Run ETO Newsletter.bat 더블클릭 — 업데이트 없이 바로 실행합니다 (인터넷이 안 될 때).

3. 최초 설정 (Settings)

프로그램 오른쪽 위 Settings 버튼에서 설정합니다. 모든 설정은 프로그램 폴더의 config.json 파일에 저장되어 다음 실행 때도 유지됩니다.

항목내용
API keyMailerLite 접속 열쇠. MailerLite 웹사이트 → Integrations → API에서 발급받아 붙여넣습니다. 입력 후 Test connection으로 "Connection OK"를 확인하세요.
Sender name / email받는 사람에게 표시될 발신자 이름과 주소 (예: hknewsletter_korea@hketotyo.gov.hk).
Subscriber group명단 대조와 발송 후 정리에 사용할 MailerLite 그룹 이름. 기본값은 ETO Korea Newsletter Subscribers 이며, MailerLite에 있는 그룹 이름과 같아야 합니다.
Footer banner links이메일 맨 아래 배너에 들어갈 홈페이지·SNS 주소 (Website, Facebook, X, Instagram, YouTube, LinkedIn). 채운 항목만 배너에 표시됩니다.
Skip SSL verification사내망 보안장비 때문에 연결이 막힐 때만 임시로 체크합니다. 평소에는 꺼 두세요.

4. 매달 발송하는 순서

  1. ① 명단 열기 — Upload list… 버튼으로 이번 달 구독자 CSV 파일을 선택합니다. 표에 명단이 나타나고 위쪽에 총 인원이 표시됩니다. (한국어 Excel로 저장한 CSV도 그대로 열립니다.)
  2. ② 명단 대조 — Compare with MailerLite 버튼을 누르면 지난달 발송 명단과 비교하여 새로 들어온 사람 / 빠진 사람 / 수신거부한 사람을 보여줍니다. 확인하고 닫으면 체크리스트에 ✓가 켜집니다. (활용법은 5장)
  3. ③ 뉴스레터 파일 선택 — Browse… 버튼으로 이번 호 Word 파일(.docx)을 선택하고, 홈페이지에 올려 둔 PDF 주소를 PDF link 칸에 붙여넣습니다.
  4. ④ 미리보기 — Preview 버튼을 누르면 실제 발송될 이메일이 브라우저로 열립니다. 제목 띠·본문·PDF 버튼· 하단 배너까지 확인하세요.
  5. ⑤ 제목 확인 — Subject 칸의 제목을 이번 달에 맞게 수정합니다 (지난번 제목이 남아 있습니다).
  6. ⑥ 테스트 메일 — Test email(s) 칸에 확인용 이메일 주소를 넣고 Send test. 여러 명이면 쉼표(,)로 이어 적습니다. 받은 메일을 컴퓨터와 휴대폰에서 열어 확인하세요.
  7. ⑦ 확인 체크 — ②와 ⑥이 모두 끝나면 "I reviewed the comparison and the test email" 체크박스가 활성화됩니다. 체크해야 발송 버튼이 눌립니다.
  8. ⑧ 발송
    • Send to all subscribers 클릭 → 표의 전원이 선택됩니다. 이번에 제외할 사람은 체크를 해제하세요.
    • 같은 버튼(Confirm — send now)을 한 번 더 클릭 → 확인 창에서 Yes.
    • 버튼이 빨갛게 바뀌며 5초 카운트다운이 시작됩니다. 취소하려면 그 빨간 버튼을 누르세요. 5초가 지나면 발송됩니다.
발송이 끝나면 "The newsletter has been sent." 안내가 뜨고, MailerLite의 구독자 그룹이 이번 발송 명단으로 자동 정리되어 다음 달 대조의 기준이 됩니다.

5. 명단 대조 창 활용하기

항목뜻할 수 있는 일
New sign-ups
(신규)		이번 CSV에 새로 들어온 사람 체크 후 Add checked to “ETO Korea Newsletter Subscribers” — MailerLite 그룹에 미리 등록합니다.
Departed
(빠짐)		지난달에는 있었는데 이번 CSV에서 빠진 사람 실수로 빠진 경우: 체크 후 Add checked to the current list — 명단에 복원되고 수정된 CSV를 저장할 수 있습니다.
정리할 경우: 체크 후 Delete checked from MailerLite… — 계정에서 삭제합니다 (수신거부자는 기록 보존을 위해 삭제되지 않습니다).
Unsubscribed
(수신거부)		이메일의 수신거부 링크를 직접 누른 사람 확인용 정보입니다. CSV에 남아 있어도 자동으로 발송에서 제외됩니다. 본인 동의 없이 다시 수신 상태로 만들지 마세요.

한 번에 한 가지 작업을 하고 창이 닫힙니다. 다른 작업이 더 필요하면 Compare with MailerLite를 다시 누르세요.

6. 뉴스레터 Word 파일 작성 요령

  • 제목은 글자 크기를 직접 키우지 말고 Word의 스타일 갤러리에서 "제목 1", "제목 2"를 지정하세요. 그래야 이메일에서도 제목 서식으로 표시됩니다.
  • 굵게, 기울임, 글머리 기호(•) 목록, 표는 이메일에 그대로 반영됩니다.
  • 문서에 붙여넣은 사진·그림은 Gmail·Outlook 등에서 표시되지 않는 경우가 많습니다. 그림이 감지되면 프로그램이 경고를 띄웁니다. 그림이 꼭 필요하면 홈페이지에 올린 뒤 링크로 넣고, 테스트 메일에서 반드시 확인하세요.
  • 상단 남색 제목 띠, PDF 다운로드 버튼, 하단 SNS·수신거부 배너는 프로그램이 자동으로 붙입니다. 본문 내용만 작성하면 됩니다.
  • Markdown(.md) 파일도 같은 방법으로 사용할 수 있습니다.

7. 메시지별 대처 방법

상황대처
발송 버튼이 회색이고 안 눌림정상 동작입니다. 명단 대조(②)와 테스트 메일(⑥)을 마치고 확인 체크박스(⑦)를 켜면 활성화됩니다.
테스트 ✓가 갑자기 꺼짐테스트 이후 Word 파일·제목·PDF 링크가 바뀌었기 때문입니다. 바뀐 내용으로 테스트 메일을 다시 보내면 됩니다.
"The PDF link does not work"PDF 주소가 틀렸거나 아직 홈페이지에 업로드되지 않은 것입니다. 주소를 고치거나 업로드 후 다시 시도하세요. 아무것도 발송되지 않았습니다.
"Possible duplicate send" (경고)같은 제목으로 최근 24시간 안에 발송된 기록이 있습니다. 이미 보낸 게 아닌지 MailerLite 발송 내역을 확인하고, 중복이 아니라고 확실할 때만 Yes를 누르세요.
"SAFETY STOP" (에러)수신자 그룹 설정이 확인되지 않아 프로그램이 발송을 중단한 것입니다(전체 오발송 방지 장치). 발송되지 않았습니다. MailerLite 대시보드에서 남아 있는 임시(draft) 캠페인을 지우고 다시 시도하세요.
"Authentication failed" (401)API 키가 비었거나 틀렸습니다. Settings에서 키를 다시 붙여넣고 Test connection으로 확인 후 저장하세요.
"Network error" / 인증서(SSL) 오류인터넷 연결을 확인하세요. 사내망에서 인증서 오류가 계속되면 Settings → Skip SSL verification을 임시로 켜면 연결됩니다.
"rate-limiting" 관련 메시지짧은 시간에 요청이 많았던 것입니다. 1분 정도 기다렸다가 다시 시도하세요.
업데이트(bat) 실패인터넷 연결을 확인하고 다시 실행하세요. 그래도 안 되면 Run ETO Newsletter.bat로 현재 버전을 그대로 사용할 수 있습니다.
그룹을 못 찾았다는 안내와 함께 계정 전체와 비교됨 Settings → Subscriber group의 이름이 MailerLite의 그룹 이름과 다른 경우입니다. 이름을 맞춰 저장한 뒤 다시 Compare 하세요.

8. 주의사항

  • 발송은 되돌릴 수 없습니다. 확신이 없으면 테스트 메일을 한 번 더 보내 확인하세요.
  • 구독자 CSV와 config.json(API 키 포함)은 개인정보·비밀 파일 입니다. 이메일·메신저로 보내거나 공유 폴더에 두지 마세요.
  • 프로그램 폴더 안의 프로그램 파일은 수정·삭제하지 마세요. (CSV 파일을 넣고 빼는 것은 괜찮습니다.)
  • MailerLite 웹사이트에서 구독자 그룹을 직접 삭제하지 마세요 — 다음 달 명단 대조 기준이 사라집니다.
